# Azure Web Farm

## About
The Azure Web Farm allows you to use a Web Role and deploy multiple web sites via MsDeploy to it. It also allows you to run lightweight console applications in the background if they are deployed alongside your websites according to a particular convention.

For more information see: http://github.com/robdmoore/AzureWebFarm

## Version 0.9.2.X breaking changes

There are a number of breaking changes that have been introduced in this version of AzureWebFarm (since v0.9.1.3). For upgrade instructions please refer to the breaking changes file at: https://github.com/robdmoore/AzureWebFarm/blob/master/BREAKING_CHANGES.md.
